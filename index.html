Типи селекторів

Селектор - це частина CSS-правила, яка повідомляє браузеру, до якого елементу веб-сторінки буде застосований CSS стиль.

Тобто, селектор - це вибірка та формальний опис того елемента чи групи елементів, до яких будуть застосовані CSS стилі.

  1. Селектори елементу

    Як селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.

  2. За id(ідентифікатор)

    Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього черезскрипти. На відміну від класів ідентифікатори мають бути унікальними, іншими словами, зустрічатися в коді документа тільки один раз.
    Звернення до ідентифікатора відбувається за допомогою атрибута id, значенням якого виступає ім'я ідентифікатора (id="hell"). Символ решітки при цьому вже не вказується.

  3. Селектори класу (.class)

    Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. мена класів повинні починатися з латинського символу і можуть містити в собі символ дефіса (-) і підкреслення (_). Використання українських літер в іменах класів неприпустимо.

  4. Селектори мультикласів(.class1.class2)

    До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу. Оскільки під час додавання кількох класів вони можуть містити однакові стильові властивості, але з різними значеннями, то береться значення в класу.

  5. Універсальний селектор ∗

    Встановдює одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт або накреслення тексту. У цьому випадку допоможе універсальний селектор, який відповідає будь-якому елементу веб-сторінки.
    Для позначення універсального селектора застосовується символ зірочки (∗). У деяких випадках вказувати універсальний селектор не обов'язково. Так, наприклад, записи ∗.class і .class є ідентичними за своїм результатом.

  6. Вкладені селектори

    Під час створення веб-сторінки часто доводиться вкладати одні елементи всередину інших. Щоб стилі цих елементів використовувалися коректно, допоможуть вкладені селектори.
    Синтаксис
    батьківський елемент дочірний елемент { Опис правил стилю }
    Не обов'язково має бути два елементи, допускається довільний рівень вкладення. Kонструкція може записуватися так: div div ul li {...}.

  7. Дочірні селектори ›

    Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елемента. Який елемент виступає батьківським, а який його нащадком легко з'ясувати за допомогою дерева елементів - так називається структура відносин елементів документа між собою.

  8. Сусідні селектори +

    Сусідні називаються елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа.
    Для керування стилем сусідніх елементів використовується символ плюса (+), який встановлюється між двома селекторами. Пробіли навколо плюса не є обов'язковими.

  9. Споріднені селектори ∼

    Споріднені селектори за своєю поведінкою схожі на сусідні селектори, але на відміну від них стильові правила застосовуються до всіх прилеглих елементів.
