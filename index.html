Основні команди git

  • git status- використовується щоб перевірити поточний стан репозиторію
  • git add-додати зміни до репозиторію
  • git commit -m "Name of the commit"-робить коміт того, що ми проіндексували, до репозиторію
  • git reset-даєможливість зняти індексацію щойно внесенних змін
  • git commit -m "Name of the commit"-робить коміт того, що ми проіндексували, до репозиторію
  • git push-відправляємо зміни в репозиторій
  • git pull-вилучаємо зміни із загального репозиторію
  • git checkout -b style-створення нової гілки з іменем style
  • git checkout-перехід між гілками
  • git branch-показує список віток, які знаходяться в поточному репозиторію
  • git log-Отримання списку змін
  • git remote -v-Додає чистий репозиторій як віддалений репозиторій до нашого оригінального репозиторію.
  • git reset-очищує буферну зону від змін, які ми щойно проіндексували.
  • git hist-робить швидку перевірку нашої історії коммітів
  • git merge-виконує злиття. Переносить зміни з двох гілок в одну
